jQuery Typography Troll

A jQuery plugin for confusing typography geeks

Is this Helvetica? Or Arial? If you aren't sure, there are plenty of resources on the interwebz to help you figure it out, and even more vitriol explaining why it matters.

Inconclusive results? Here's some more sample text, in large print for your viewing pleasure:

Interrupt aardvark JUGGLE ROCCOCO flitter

If you've got both fonts on your system (sorry, Windows users), it's obviously both. Check out the README and start confusing typographers today.
